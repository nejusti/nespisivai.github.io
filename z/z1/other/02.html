Пример 02

Обьект - это единая конструкция, содержашая в себе описания
данных и функций или, в терминологии JavaScript, свойств и
методов.

Программист может создать экземпляр обьекта при помощи
команды new Созданный экземпляр будет обладать свойствами и
методами обьекта, от которого он был создан.

Создадим экземляр now от встроенного объекта Date и
вычислим с помощью него число дней, прошедших с начала года:

// Текущая дата и время.
var now = new date();

// Начало текущего года.
var begin = new Date(now.getYear(),0,1);

// Число в милисикундах от начала года.
var d = now.getTime() - begin.getTime();

// Число милисекунд в дне.
var msPerDay = 24*60*60*1000;

// Поделим на количество миллисекунд в сутках.
d /= msPerDay;

// Покажем результат.
alert("Прошло " + Math.Floor(d)) + " дней с начала года";
